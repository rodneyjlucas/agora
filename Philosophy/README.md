# Stoic Philosophy Library

A curated collection of public-domain English translations of the three great Roman Stoics: **Marcus Aurelius**, **Epictetus**, and **Seneca the Younger**. All texts are sourced from Project Gutenberg.

## How to populate this folder

Open Terminal in this folder and run:

```bash
bash download_stoics.sh
```

The script creates three subfolders (`Marcus Aurelius - Meditations/`, `Epictetus/`, `Seneca/`) and fills them with the files listed below. It skips anything already downloaded, so it's safe to re-run.

## Reading order (if you want one)

A gentle on-ramp into Stoic thought:

1. **Epictetus — *Enchiridion*** (short, punchy, the "handbook"). Start with Carter's translation.
2. **Marcus Aurelius — *Meditations*** (private journal, aphoristic). Long is readable; Casaubon is more archaic and beautiful.
3. **Seneca — *Moral Letters*** (essays disguised as letters to a friend). Start with Vol. 1 and dip in.
4. **Epictetus — *Discourses*** (longer lecture transcripts, deeper).
5. **Seneca — *Minor Dialogues*** for *On the Shortness of Life* and *On Anger*.

All files are plain UTF-8 text.
